WP Location Redirect is a powerful WordPress plugin that enables you to redirect users to a specific URL based on their geographical location. Whether by country, state/region, or city, this plugin utilizes the GeoIP2-City database to accurately determine where your visitors are coming from.

== Description ==

With WP Location Redirect, you can easily set up location-based redirection rules for your users. It's a lightweight, high-performance solution for tailoring content delivery based on user location.

Features include:
* **Country-Level Redirects**: Redirect users based on their country codes.
* **State/Region-Level Redirects**: Set up redirects for specific states or regions.
* **City-Level Redirects**: Enable more granular redirects for targeted cities.
* **Admin Interface**: Manage redirection rules via an intuitive dashboard.
* **GeoIP2 Database Integration**: Rely on accurate MaxMind GeoIP2 data.
* **Automated Updates**: Ensure continuous accuracy with easy database updates.

== Installation ==

1. Download and upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin from the **Plugins** screen in WordPress.
3. Use the settings interface to configure your location-based redirect rules.

== Frequently Asked Questions ==

= What are the system requirements? =
WordPress version 5.6 or higher and PHP version 7.4 or greater.

= How do I update the GeoIP2 database? =
You can upload an updated `GeoIP2-City.mmdb` file through the admin interface.

= Does this plugin collect user data? =
No, the plugin only utilizes GeoIP2 data and does not collect or store any user-specific information.
